# CodePulse 开发说明

## 环境要求

1. Windows 10 或 Windows 11。
2. Node.js 与 pnpm。
3. 统一使用 pnpm，不使用 npm、npx 或 yarn。

## 常用命令

```powershell
pnpm install
pnpm run typecheck
pnpm run test
pnpm run build
pnpm run start
```

## 当前 MVP 范围

当前代码完成首批基础能力：

1. Electron、Vue 3、TypeScript strict、Vite、Pinia、Vue Router、Element Plus、ECharts、SQLite 依赖和 Electron Builder 配置。
2. 共享模型、适配器接口、IPC channel 与设置模型。
3. MockAdapter 与 AgentStateHub。
4. 托盘、Explorer 重启恢复、系统休眠恢复、动态岛窗口、显示器变化处理、任务栏贴边弹窗和任务中心交互补强最小版本。
5. CodexAdapter 进程检测、可配置状态源解析、可配置日志源解析、状态源和日志源多字段会话识别、设置页状态源和日志源路径入口、状态源和日志源路径运行期热更新，ProcessAdapter 本机 Agent 进程检测最小版本，LogAdapter 通用 UTF-8 JSONL 日志源解析和日志源路径运行期热更新最小版本，CustomCommandAdapter 默认禁用、授权执行和运行期配置同步最小版本，以及 `providers.detect()` 真实检测链路。
6. IPC 参数校验、安全测试、任务操作授权和错误码归一化最小版本，包含数据源启用与设置持久化联动、`settings.update` 启用状态同步 AgentStateHub、Codex/通用日志路径运行期同步、自定义命令授权/命令路径/参数运行期同步、任务打开、打开 Codex Agent、非 Codex Agent 项目目录回退、复制摘要、稍后提醒和标记已读的真实任务校验，Preload API 白名单冻结、状态订阅清理和诊断导出脱敏。
